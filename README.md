# Bluetooth Relay Control With SoftwareSerial
Arduino Bluetooth Relay Control With SoftwareSerial.

## Prerequisites
1. Arduino Uno R3
2. HC-05 / HC-06 Bluetooth Module
3. 2 Channel Relay Module
4. Bulb
5. Bulb Holder
6. Wire
7. Plug
8. Jumper Wires
